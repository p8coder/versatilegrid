Requirements

There are three requirements you need to meet before you can use versatileGrid: jquery 1.10 +, bootstrap and font-awesome.


Installation

Add css files in the header section


 <link href="your_css_path/bootstrap.min.css" rel="stylesheet">
 <link href="your_css_path/font-awesome.min.css" rel="stylesheet">
 <link href="your_css_path/versatileGrid.css" rel="stylesheet">

Add js at the bottom closest to </body> tag


<script src="your_js_path/jquery-1.10.1.min.js" type="text/javascript"  ></script>
<script src="your_js_path/versatileGrid.js" type="text/javascript" ></script>
<script src="your_js_path/bootstrap.min.js" type="text/javascript" ></script>

bootstrap.min.js file is optional, add it only if you are going to use functions or plugins from bootstrap, otherwise is no needed.


Basic Setup

If you are going to show static information, you can use the following HTML code. (This is default HTML for tables in Bootstrap)


<table class="table" id="vgTable" data-title="Example Users" data-colsWidth="30" >
	<thead>
		<tr >
			<th >Id</th>
			<th >First name</th>
			<th >Last name</th>
			<th >Phone</th>
			<th >Username</th>
		</tr>		
	</thead>
	<tbody>
		<tr>
			<td>1</td>		
			<td>Mark</td>
			<td>This is a long long long long very very very long so long example text text text and text</td>
			<td>315-789-8038</td>
			<td>mSmith</td>
		</tr>
		<tr>
			<td >2</td>		
			<td>Sean</td>

Run versatileGrid

$(function() {
	$("#vgTable").versatilGrid();
});

If you want to apply versatilGrid to multiple grid on the fly, you can set different styles via attributes. Reffer to basic_multiple.html file in examples.



Options

Options can be passed via data attributes for static data tables, for dynamic data use JavaScript options.

Name type default description
title string Grid Name Add a Title to Grid Header. Available via attributes for static data: data-title
theme string gray Change the style for Grid. Available themes: gray, blue, green, orange, dark. Available via attributes for static data: data-theme
colswidth array [ ] Array for column widths in pixels. First index in the array is gonna be the width for first column, second index will be width for second column...etc. You can use auto for not apply width.
Available via attributes: data-colswidth

Some Examples:
  • [30] : Width for first column only.
  • [auto, auto, 150] : Width for third column only.
  • [30, auto, 150] : Width for first & third column only.
limits array [10,20,30] This array hold the limit of rows per page. You will see a dropdown select with these limits down right in the Grid Footer.
defaultLimit int 10 This going to be the default selected limit from previous option (limits). This value must exists in limits array.
url string If a url script is provided, content will be loaded via jQuery's Ajax method and injected into the grid. Ther response from script should be in Json Format.
onLoadData function This will trigger the function passed each time the grid load data pages.
onBeforeLoadData function This will trigger the function passed before each time the grid load data pages.
checkbox boolean false This will enable multiple selector by checkboxes in each row.
toolbar Array This will enable the toolbar section. Bellow is an example to add buttons to toolbar: 
$("#myGrid").versatileGrid({
   title : 'Contacts',
   toolbar : [
 	{type:'button', name:'Activate', icon:'icon-ok', click:activateUsers},
	{type:'button', name:'Add New', icon:'icon-plus', click:addNew}
   ]
});

You need to have defined functions to use them in the click event.
errorMsg String Sorry, there was a problem getting the data. This message will be shown if there is a problem getting the information from script.
notFoundMsg String Sorry, The server script was not found. This message will be shown if the server script doesn't exits.

Methods

.versatileGrid('getSelectedRows')

Return an array with the row ids selected.

$("#myGrid").versatileGrid('getSelectedRows')

.versatileGrid('refreshGrid')

Reload grid using default or user options. You can use it for example after add or delete a record from the table.

$("#myGrid").versatileGrid('refreshGrid')

Dynamic Data

 	 "id":"15",
 	 "data":["15","Katty","Phillips","qwerty@sdfgf.com","8545 5425 544"],

 ]
